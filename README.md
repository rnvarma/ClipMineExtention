ClipMineExtention
=================
HOW TO INSTALL:

- go to: chrome://extensions/
- click on the "Load unpacked extensions" button
- select the directory where you pulled the repository from

HOW TO USE:
- go to youtube pages or google or facebook and if there are videos on the page to watch, 
you will have the option to open it in the clipmine interface
- click on the button on the top right and enjoy!
